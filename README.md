## Gestion des commits et des versions

Nous utilisons le système Conventional Commits pour nommer nos commits, et le versioning sémantique (SemVer) pour numéroter les versions.  
Chaque commit doit refléter clairement la nature du changement, et les tags Git identifient les versions publiées.
